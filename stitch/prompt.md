# Digital Designer Portfolio UI Prompt

Create a modern, premium personal portfolio website for a digital designer.

## Style & Aesthetics
- **Theme**: Dark mode with velvet/slate backgrounds.
- **Typography**: Modern sans-serif (e.g., Inter, Outfit, or Poppins). Large, bold headings.
- **Visuals**: Glassmorphism effects, subtle gradients, and large artwork previews.
- **Animations**: Smooth scroll, hover effects, and micro-animations for interactivity.

## Sections to Include
1. **Hero**: Catchy headline, brief introduction, and a high-quality visual or 3D element.
2. **About**: Professional bio detailing design philosophy and experience.
3. **Portfolio Gallery**: A masonry or grid layout with large cards for design projects. Each card should have a hover effect showing the title and category.
4. **Skills**: Visual representation of skills (e.g., icons or progress bars) using a modern layout.
5. **Contact**: Minimalist form with fields for Name, Email, and Message. Include social media links.
6. **Footer**: Simple copyright information and quick links.

## User Experience
- Ensure the layout is clean and minimal.
- Prioritize high-impact imagery.
- The design should feel "alive" through subtle motion.
